# PLAB Check

PLAB 매치 링크를 붙여 넣으면 경기 정보와 신청자 현황을 보여주는 개인용 도구입니다.

## 바로 사용하기

**[PLAB Check 사이트 열기](https://plab-checker.funfun0218.chatgpt.site)**

현재 사이트는 `abr.ge` 짧은 링크를 서버에서 안전하게 확인한 뒤 PLAB 매치 API를 조회합니다.

## 기능

- `https://abr.ge/vjss2z` 형식의 짧은 링크 입력
- 매치 번호 자동 추출 및 경기 정보 조회
- 확정·취소·전체 신청자 필터
- 정원, 참가비, 구장, 레벨, 플레이스타일 표시
- 날짜·지역별 여성 매치 조회

## 로컬 실행

```bash
npm install
npm run dev
```

빌드 확인:

```bash
npm run build
```
